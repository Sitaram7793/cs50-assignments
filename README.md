# CS50-assignments

This repository contains CS50 Scratch assignments and mini-projects.

## Week0 - Scratch Project
  **Bottle Shooter** — folder: `bottle-shooter/`  
  Short description: A simple arcade game built in Scratch.

  ## How to run
  1. Download `.sb3` file from the project folder.
  2. Open on scratch.mit.edu (Choose File → Upload from your computer) or use the Scratch offline editor.
